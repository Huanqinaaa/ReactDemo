## install and run

clone

```
> $ git clone https://github.com/Huanqinaaa/ReactExample
```

npm install under react and server catlog

```
> $ npm install
```

start backend under server file

```
> $ npm run start
```

start webpake under react file

```
> $ npm run dev
```

release project commmand

```
> $ npm run build
```
